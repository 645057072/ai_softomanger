# 生成式AI工程师理论模拟题 - 数据库表设计

## 一、题目表 (questions)

| 字段名 | 数据类型 | 是否必填 | 说明 |
|--------|----------|----------|------|
| id | INT | 是 | 主键，题目ID，自增 |
| exam_type | VARCHAR(50) | 是 | 试卷类型，如：高级、中级、初级 |
| exam_subject | VARCHAR(100) | 是 | 试卷科目，如：生成式AI工程师 |
| question_type | VARCHAR(20) | 是 | 题目类型：single_choice(单选题)、multiple_choice(多选题)、judgment(判断题) |
| question_no | INT | 是 | 题号 |
| question_text | TEXT | 是 | 题目内容 |
| option_a | VARCHAR(500) | 否 | 选项A内容（单选题/多选题必填） |
| option_b | VARCHAR(500) | 否 | 选项B内容（单选题/多选题必填） |
| option_c | VARCHAR(500) | 否 | 选项C内容（单选题/多选题必填） |
| option_d | VARCHAR(500) | 否 | 选项D内容（单选题/多选题必填） |
| correct_answer | VARCHAR(50) | 是 | 正确答案：单选题(A/B/C/D)、多选题(ABC/A,C等)、判断题(正确/错误) |
| explanation | TEXT | 否 | 答案解析 |
| score | DECIMAL(5,2) | 是 | 题目分值，按照题目类型自定义 |
| page_number | INT | 否 | 来源页码 |
| created_at | DATETIME | 是 | 创建时间 |
| updated_at | DATETIME | 是 | 更新时间 |

## 二、字段含义详解

### exam_type（试卷类型）
- 试卷的等级或类型，如：高级、中级、初级等

### exam_subject（试卷科目）
- 试卷所属的科目，如：生成式AI工程师、机器学习工程师等

### question_type（题目类型）
- `single_choice`：单选题，包含4个选项(A/B/C/D)，只有一个正确答案
- `multiple_choice`：多选题，包含4个选项(A/B/C/D)，有一个或多个正确答案
- `judgment`：判断题，答案为"正确"或"错误"

### question_no（题号）
- 题目在试卷中的序号，如1、2、3...200

### question_text（题目内容）
- 题目的具体问题描述

### option_a / option_b / option_c / option_d（选项内容）
- 单选题和多选题需要填写
- 判断题此四字段为空

### correct_answer（正确答案）
- 单选题：A、B、C、D 其中之一
- 多选题：多个选项组合，如"ABC"、"A,C"、"AC"等格式
- 判断题：正确、错误

### explanation（答案解析）
- 对正确答案的详细解释说明

### score（题目分值）
- 按照题目类型自定义的分值标准
- 例如：单选题1分、多选题2分、判断题0.5分等

### page_number（来源页码）
- 题目在PDF中的页码位置

## 四、建表SQL语句

```sql
CREATE TABLE questions (
    id INT PRIMARY KEY AUTO_INCREMENT,
    exam_type VARCHAR(50) NOT NULL COMMENT '试卷类型：高级/中级/初级',
    exam_subject VARCHAR(100) NOT NULL COMMENT '试卷科目：生成式AI工程师等',
    question_type VARCHAR(20) NOT NULL COMMENT '题目类型：single_choice/multiple_choice/judgment',
    question_no INT NOT NULL COMMENT '题号',
    question_text TEXT NOT NULL COMMENT '题目内容',
    option_a VARCHAR(500) DEFAULT NULL COMMENT '选项A',
    option_b VARCHAR(500) DEFAULT NULL COMMENT '选项B',
    option_c VARCHAR(500) DEFAULT NULL COMMENT '选项C',
    option_d VARCHAR(500) DEFAULT NULL COMMENT '选项D',
    correct_answer VARCHAR(50) NOT NULL COMMENT '正确答案',
    explanation TEXT DEFAULT NULL COMMENT '答案解析',
    score DECIMAL(5,2) NOT NULL COMMENT '题目分值',
    page_number INT DEFAULT NULL COMMENT '来源页码',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    INDEX idx_exam_type (exam_type),
    INDEX idx_exam_subject (exam_subject),
    INDEX idx_question_type (question_type),
    INDEX idx_question_no (question_no)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='生成式AI工程师理论模拟题表';
```
