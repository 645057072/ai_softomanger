## 技术实现

### 文件结构
```
frontend/src/
├── App.vue                          # 主应用组件（支持布局切换）
├── components/
│   └── Layout.vue                  # 系统导航布局（上中下结构）
├── views/
│   ├── auth/
│   │   ├── Login.vue              # 登录页面
│   │   └── Register.vue           # 注册页面
│   ├── Home.vue                   # 首页
│   ├── BI.vue                     # 驾驶 BI
│   ├── Study.vue                  # 学习中心
│   ├── Paper.vue                  # 试卷中心
│   ├── Exam.vue                   # 考试中心
│   └── System.vue                 # 系统设置
└── router/
    └── index.js                   # 路由配置
```

### 路由配置
```javascript
/              → 重定向到 /home
/home          → 首页（驾驶 BI）
/login         → 登录页面
/register      → 注册页面
/bi            → 驾驶 BI
/study         → 学习中心
/paper         → 试卷中心
/exam          → 考试中心
/system        → 系统设置
```

### 布局切换逻辑
- 未登录：显示默认首页
- 访问 /login 或 /register：显示认证布局（居中卡片）
- 已登录：显示系统导航布局（上中下结构）

## 部署步骤

### 本地开发
```bash
cd frontend
npm install
npm run dev
```

### 生产构建
```bash
cd frontend
npm run build
```

### Docker 部署
```bash
# 在服务器上
cd /opt/ai_softomanger
docker-compose down
docker-compose up -d --build
```
