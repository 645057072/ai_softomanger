# 考试系统前端项目

## 项目结构

```
frontend/
├── public/
│   └── index.html
├── src/
│   ├── assets/
│   ├── components/
│   ├── views/
│   ├── router/
│   ├── api/
│   ├── store/
│   ├── utils/
│   ├── App.vue
│   └── main.js
├── package.json
└── vite.config.js
```

## 技术栈
- Vue 3
- Element Plus
- Axios
- Vue Router
- Pinia
- Socket.io-client

## 安装依赖
```bash
npm install
```

## 开发环境启动
```bash
npm run dev
```

## 构建生产版本
```bash
npm run build
```
